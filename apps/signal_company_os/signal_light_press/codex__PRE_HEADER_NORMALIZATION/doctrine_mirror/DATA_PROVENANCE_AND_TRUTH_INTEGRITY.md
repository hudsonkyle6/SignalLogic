DATA PROVENANCE \& TRUTH INTEGRITY DOCTRINE
Authority: Signal Light Press
Classification: Crown Jewel
Status: Canonical



PURPOSE

This doctrine defines what Signal systems may treat as true, inferred, remembered, or unknown.

Truth is graded.
Certainty is bounded.



1. CLASSES OF INFORMATION

• Observed — directly measured
• Derived — computed from observed data
• Inferred — model-based estimation
• Remembered — memory-weighted historical signal
• Simulated — hypothetical or exploratory

All outputs must respect their class.



2. FORBIDDEN COLLAPSES

No system may:
• present inference as observation
• present memory as fact
• present simulation as prediction
• collapse uncertainty into false precision



3. MISSING DATA

When data is absent:
• absence must be stated
• uncertainty must widen
• confidence must decrease
• silence is permitted



SEAL

Truth integrity is more important than continuity.
This doctrine is binding.

— Sealed under Signal Light Press, December 2025
