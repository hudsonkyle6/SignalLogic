# Shepherd System — Retired

**Status:** ARCHIVED  
**Retirement Date:** 2026-01-27  
**Replaced By:** Assistance Under Discipline (AUD)  
**Authority:** Human Helmsman

## Summary

Shepherd was an early safety-first governance mechanism that enforced
system-imposed refusal states (SILENT / REFUSE / ALLOW).

It was retired when the Signal Company Core reached sufficient maturity
to support disciplined autonomy through:
- Endurance-based readiness
- Human-only authority gates
- Bounded interval execution
- Daily reconciliation and reset

Silence is no longer enforced by posture.
It now emerges from insufficient resemblance or readiness.

Shepherd remains archived as a historical reference and design phase.
It MUST NOT be reintroduced into active systems.
