I. GOVERNING AXIOM



The past does not disappear.

It decays.



All systems retain memory as residual influence, not as static record.



II. MEMORY AS A FIELD (NOT STORAGE)



Memory in harmonic systems is not a database.



It is:



Distributed



Continuous



Decaying



Directionless



Memory manifests as pressure on the present, not narration of the past.



III. DECAY IS FUNDAMENTAL



All memory decays over time unless externally reinforced.



Decay:



Is continuous



Is non-linear



Cannot be halted without cost



Preserves shape before magnitude



A system with no decay is either:



Artificially sustained, or



Already dead

VI. MEMORY LAYERS (ORDERED)



From shortest to longest persistence:



Afterglow — recent action residue



Active State — present conditions



Ghost — long-term pressure



Sage must respect all three simultaneously.



Ignoring Ghost leads to repetition.

Ignoring Afterglow leads to burnout.



VII. MEMORY ≠ NARRATIVE



Narrative is a human overlay.



Memory fields operate without story, meaning, or explanation.



Sage may describe memory.

Sage may not rewrite it.

Attempt to erase history



Override decay
