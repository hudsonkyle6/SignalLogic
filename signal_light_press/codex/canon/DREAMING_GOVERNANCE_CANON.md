I. DREAMING IS A POSTURE, NOT A PROCESS



Dreaming is a non-operational system state in which:



Action is suspended



Learning is disabled



Memory is pruned



Confidence decays



Identity stabilizes



Dreaming exists to prevent continuous cognition, which is a known collapse mode.



II. DREAMING ENTRY CONDITIONS (ANY ONE SUFFICIENT)



Dreaming must be entered when any of the following are true:



Post-Action Cooldown



Any non-trivial action window has closed



Capital, deployment, or narrative energy was expended



Drift Accumulation



Drift index rising without matching environmental pressure



Indicates internal bias accumulation



Low-Volatility Plateau



Stable external conditions + internal activity persistence



Indicates unnecessary motion



Seasonal Boundary Crossing



Completion of a seasonal or operational arc



Buffer Thinning



Stabilizing capacity approaches disturbance pressure



Shepherd is the final authority to enter Dreaming.



III. DREAMING STATE CONSTRAINTS (ABSOLUTE)



While Dreaming is active:



Function	Status

Oracle ingestion	PAUSED

Sage narration	READ-ONLY

Shepherd permissions	HOLD

Learning	DISABLED

Optimization	FORBIDDEN

Forecasting	FORBIDDEN

Justification	FORBIDDEN



No subsystem may override Dreaming.



IV. PERMITTED OPERATIONS DURING DREAMING



Only the following are allowed:



Memory Decay \& Pruning



Afterglow damping



Low-utility memory collapse



Redundant signal dissolution



Drift Recalibration



Widening confidence bands



Punishing false certainty



Ghost Recompression



Identity compression



No narrative retention



Intensity Forgetting



Recent success and failure lose weight



V. DREAMING EXIT CONDITIONS (ALL REQUIRED)



Dreaming may only exit when all are true:



Drift stabilized or declining



Buffers rebuilt above minimum thresholds



Confidence bands normalized



No unresolved boundary alerts



Shepherd explicitly permits wake-up



No Engine may self-wake.



VI. FAILURE MODE PREVENTION



Dreaming prevents:



Overlearning



Reflexive optimization



Momentum addiction



Narrative self-deception



Sleep is governance.



CANONICAL SUMMARY



Dreaming is the Signal System’s refusal to mistake motion for intelligence.
