Theory Only · No Math · No Code · No Tasks



This course defines how Shepherd resolves conflicts between inputs.

It does not compute values, thresholds, or outcomes.



It establishes priority law.



1\. Purpose



Shepherd receives multiple informational inputs that may disagree.



Course III-C answers:



When signals conflict, which ones dominate?



This prevents:



oscillation



justification hacking



optimistic overrides



accidental escalation



2\. Inputs Shepherd May See (Read-Only)



Constraint envelopes (thermal, seasonal, structural, energetic, etc.)



Antifragile indices (unknowns, drift, brittleness, strain)



Oracle phase + confidence (if present)



Shepherd never sees:



raw data



tasks



user intent



schedules



goals



3\. Canonical Ordering (Locked)



Shepherd resolves posture in the following strict order:



Order of Dominance

1\. Hard Constraints

2\. Antifragile Signals

3\. Oracle Phase

4\. Absence / Silence





Earlier items always override later ones.



4\. Dominance Rules (Canonical)

Rule 1 — Hard Constraint Supremacy



If any hard constraint is violated:



→ REFUSE



No further evaluation occurs.



Examples:



thermal impossibility



structural instability



legal or ethical violation



irreversibility under unsafe conditions



Rule 2 — Antifragile Suppression



If antifragile indices indicate elevated risk:



→ SILENT or REFUSE (depending on severity)



Antifragile signals bias toward restraint even when constraints technically allow.



Rule 3 — Phase Advisory Role



Oracle phase is consulted only if:



no hard constraints are violated



antifragile indices are within tolerance



Phase may:



contextualize posture



explain restraint



annotate ALLOW



Phase may never override suppression.



Rule 4 — Absence Defaults



If:



Oracle emits no phase



Confidence is low



Inputs conflict ambiguously



→ SILENT



Silence is not indecision.

It is lawful governance.



5\. Explicit Refusals (Locked)

weighted averaging

fallback actions



Resolution is ordinal, not numerical.
